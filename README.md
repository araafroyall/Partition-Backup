# Partition-Backup
A Dynamic CLI Module that Allow to Backup Android Partitions From Terminal



# Usage
```
Usage: backup [OPTIONS] [PARTITIONS]
Options:
  -h                    Show usage
  -d                    Backup path (default: /sdcard/backup)

Partition Examples:
  all
  boot[_a,_b]
  persist[_a,_b]
  system[_a,_b]
  userdata[_a,_b]
  vendor[_a,_b]
```
